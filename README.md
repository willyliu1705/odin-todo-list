# Project: Todo List

TOP's project to practice classes, import/export, npm, webpack, and the Web Storage API.

## Description

This is an unstyled to-do list that allows you to create folders and add tasks in specific folders.
- The functionality of the todo list was more important so there is almost no styling.

## Features

- can create a folder that each have their own separate tasks workspace
- can add and delete tasks
- changes are locally stored, so you don't lose progress even if the tab is closed!
